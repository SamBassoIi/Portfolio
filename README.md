# Samuel Bassoli | Portfólio Pessoal

Bem-vindo ao repositório do meu portfólio pessoal! Este projeto foi desenvolvido para apresentar minhas habilidades, projetos em destaque e meu perfil profissional como Desenvolvedor de Sistemas.

##  Tecnologias Utilizadas

Este portfólio foi construído com as seguintes tecnologias:
* **HTML5** - Estruturação semântica.
* **CSS3** - Estilização customizada com variáveis (Dark Mode), Flexbox, CSS Grid e rolagem suave.
* **JavaScript** - Integração de iframes interativos e manipulação de DOM.
* **Netlify** - Deploy e hospedagem contínua.

##  Funcionalidades em Destaque

* **Design Responsivo:** Adaptado para funcionar perfeitamente em celulares, tablets e desktops.
* **Live Preview (Iframes):** Os projetos hospedados rodam diretamente dentro do portfólio, permitindo que o usuário teste os sistemas sem sair da página.
* **Rolagem Suave:** Navegação âncora aprimorada para melhor Experiência do Usuário (UX).
